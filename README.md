# Rationale
This is a performance benchmark for [Go!AOP](https://github.com/goaop/framework) framework for Aspect Oriented Programming
in PHP.

# Usage
```bash
composer install
vendor/bin/phpbench run GoAopBench.php --report=aggregate
```
